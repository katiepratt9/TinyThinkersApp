# Tiny Thinkers App
2D educational Android game suite designed for young children, featuring six interactive mini-games that promote early learning through play. Built using Java, XML, and Android Studio, Tiny Thinkers emphasizes simple navigation, bright visuals, audio feedback, and persistent scoring to create a fun and accessible learning experience.

# Features
- Kid-friendly UI with bright colors and large buttons
- Account system with Login, Create Account, and Forgot Password
- Main menu with six mini-games:
	- Picture Alphabet
	- Hangman
	- Math Game
	- Memory Flip
	- Tic-Tac-Toe
	- Matching Game
- Individual high scores saved using SharedPreferences
- Smooth activity navigation using Intents
- Background music and sound effects
- Help screens for each game

# How to Run
1. Clone the repository: git clone https://github.com/katiepratt9/TinyThinkersApp.git
2. Open the project in Android Studio
3. Allow Gradle to sync
4. Run on an emulator or Android device

# Note
- High scores persist between sessions
- Each game includes a Help button with instructions
- No strict validation is enforced for usernames or passwords (usernames are not required to be email addresses).
- The password recovery feature requires the emulator to be logged into an email account in Android Studio to send the recovery link.     
- Back buttons return users to the main menu for easy navigation, however, there is no logout button implemented
